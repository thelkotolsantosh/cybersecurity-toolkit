INSTALLATION QUICK REFERENCE
================================================================================

1. Clone Repository:
   git clone https://github.com/yourusername/cybersecurity-toolkit.git
   cd cybersecurity-toolkit

2. Create Virtual Environment:
   python3 -m venv venv
   source venv/bin/activate

3. Install Dependencies:
   pip install -r requirements.txt

4. Verify Installation:
   python main.py --help

5. Run Tests:
   python -m pytest tests/

USAGE QUICK REFERENCE
================================================================================

Network Scan:
  python main.py --scan-network 192.168.1.0/24

Port Scan:
  python main.py --port-scan example.com --ports 1-1000

SSL Check:
  python main.py --ssl-check example.com:443

Subdomain Enumeration:
  python main.py --subdomain-enum example.com

Security Headers:
  python main.py --check-headers https://example.com

Password Strength:
  python main.py --password-strength "Password123!"

Verbose Output:
  python main.py --port-scan example.com --verbose

Save Results:
  python main.py --port-scan example.com --output results.txt
